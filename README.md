# How to run

1. Open a terminal and move into the project folder

   > cd HealtyEats

2. [Install Flask](https://flask.palletsprojects.com/en/2.0.x/installation/)

3. Set Flask app

   > set FLASK_APP=app

4. Set Flask Envirorment

   > set FLASK_ENV=development

5. Run the application

   > flask run
